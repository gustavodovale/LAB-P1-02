# Construindo o Transformer Encoder From Scratch

Este repositório contém a implementação do Forward Pass de um bloco **Transformer Encoder**, desenvolvida para fins acadêmicos. O objetivo principal é desmistificar o funcionamento interno da arquitetura proposta no artigo *"Attention Is All You Need"* (2017), utilizando exclusivamente as bibliotecas `numpy` e `pandas`.

## 1. Tecnologias Utilizadas
* **Python 3.x**
* **NumPy:** Para todas as operações de álgebra linear e manipulação de tensores.
* **Pandas:** Para simulação e visualização do vocabulário inicial.

## 3. Como Rodar o Código

### Pré-requisitos
Certifique-se de ter o Python instalado. É recomendado o uso de um ambiente virtual.

#### Crie e ative um ambiente virtual
pip install numpy pandas

#### Para roda:
executa no temrinal na pasta onde tá o arquivo: python main.py

#### Uso de IA Generativa
Uso de IA Generativa: Ferramentas de IA (Gemini/ChatGPT) foram consultadas exclusivamente para brainstorming de sintaxe do NumPy (como o uso de operadores de multiplicação de matriz @, swapaxes e regras de broadcasting para a LayerNorm).
